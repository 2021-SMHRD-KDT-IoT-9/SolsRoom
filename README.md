<h3>프로젝트 소개</h3>
<p>아두이노를 기반으로 흡연부스 내부의 기계장치를 제어하고 흡연부스에 설치된 센서(심박수,산소포화도센서,가스센서 등)의 값을 wifi통신을통해 web에서 활용하여 사용자 및 관리자에게 서비스를제공
</p>
